What's new

Secure API key authentication — no password stored
Real-time sync progress shown in the extension popup
Getting started
1
Create a free account at garminbadges.com — sign in with Google or email and password.
2
Go to your Dashboard and copy your API key.
3
Click the extension icon in your toolbar, open Settings, and paste your API key.
4
Browse to Garmin Connect, make sure you are logged in, then click Sync now.
